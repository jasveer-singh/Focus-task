# Focus Task Browser Extension

## Load in Chrome
1. Open `chrome://extensions/`
2. Enable Developer mode
3. Click "Load unpacked"
4. Select this `extension/` folder

## What it does
- Click extension button -> "Create task from page"
- Opens the app with `capture=1` URL params
- App stores it in Ideas Inbox automatically

## Configure app URL
- Edit `popup.js` and change `APP_URL` if needed.
